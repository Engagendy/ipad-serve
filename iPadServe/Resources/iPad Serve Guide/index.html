Pick any page

Your site

CSS, JavaScript, modals, videos, and relative paths work through localhost.

03

Run locally

iPad Serve starts a private server on the iPad and serves the selected files to WebKit.

Good fit

Projects that work best.

  • Static HTMLPlain `.html`, `.css`, `.js`, images, fonts, PDFs, and media files.
  • Relative pathsLinks like `assets/app.js` and `../images/photo.jpg` stay inside the folder.
  • Client-side JSButtons, tabs, charts, modals, and browser-only app logic run normally.
