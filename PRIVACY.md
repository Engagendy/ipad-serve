# Privacy Policy for HTML Serve

Effective date: May 13, 2026

HTML Serve runs static HTML projects that you explicitly import from the system Files app. The app does not require an account and does not upload your files to a server.

## Data Collection

HTML Serve does not collect, store, sell, share, or transmit personal data.

The app does not collect analytics, identifiers, diagnostics, location, contact information, financial information, browsing history, or user content.

## File Access

HTML Serve can only access folders that you choose through the system Files picker.

Imported folders are copied into the app's local storage so they can be opened offline. HTML, CSS, JavaScript, media, images, PDFs, and other assets are served locally on device through a loopback server and displayed inside the app.

Files are not uploaded to any server or shared with the developer.

## Network Use

HTML Serve starts a local loopback HTTP server on the device to display imported static projects. This local server is used only by the app on the same device.

The app does not use analytics services, advertising SDKs, tracking SDKs, payment processors, authentication providers, cloud APIs, or external backend services for its core functionality.

Imported web projects may contain their own external links or remote scripts. Those requests are controlled by the imported project content, not by HTML Serve.

## Children's Privacy

HTML Serve does not knowingly collect personal information from children.

## Changes to This Policy

This privacy policy may be updated when the app changes. Updates will be published in this repository.
